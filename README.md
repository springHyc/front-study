## npm 更新

### 安装

`npm install npm-check-updates -g`

`npm install npm-check -g`

### 检查

检查 depedencies 中的最新版本 `npm-check-updates` 或者 `ncu`

### 更新

更新 dependencies 到最新版本 `ncu -u`

`npm install` 可以将`node_modules`先删除了再继续
